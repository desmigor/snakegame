# Snake Game
This is a Nokia 3310 SNAKE game implemented in HTML,CSS and JS.
The game has the following features/functionalities:

1. Start/Restart game
2. Pause/Resume game
3. Game statuses (Ready, playing, paused and lost)
4. Score count
